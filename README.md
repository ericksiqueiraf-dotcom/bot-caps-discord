# Bot Caps Discord

Bot para organizar partidas personalizadas de League of Legends no Discord com fila persistente, consulta de elo na Riot API e balanceamento por MMR.

## Comandos

- `!entrar Nome#TAG`
- `!lista`
- `!start`
- `!reset`

## Configuracao

1. Preencha o arquivo `.env` com `DISCORD_TOKEN` e `RIOT_API_KEY`
2. Preencha o `config.json` com os IDs dos canais de voz
3. Instale as dependencias
4. Rode o bot

## Execucao

```bash
npm install
npm start
```
